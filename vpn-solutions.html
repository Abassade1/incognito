VPN Solutions

WORK

Implementation of a VPN solution using F5 technology for Airtel Nigeria

The brief

A Virtual Private Network (VPN) is used for creating a private scope of computer communications or providing a secure extension of a private network through an insecure network such as the Internet. VPN is a widely used in network security. VPN can be built upon IPSec or Secure Socket Layer (SSL).

The Work

Awareness and Consistency

Designing and implementing the SSL VPNs exemplify a number of security principles and technologies, including crypto, integrity, authentication, key management, key exchange, and Public- Key Infrastructure (PKI). To achieve this goal, we implemented a simple SSL VPN for Linux Ubuntu operating system.
